üí° What is Code?

Code = Instructions for Computers

Code is simply a way to tell computers what to do. Just like you follow a recipe to bake a cake, computers follow code to do tasks.

üé≠ Three Languages of the Web

Think of a House üè†

  • HTML = The structure (walls, doors, rooms)
  • CSS = The decoration (paint, furniture, style)
  • JavaScript = The functionality (lights turn on, doors open)

üì∫ Or Think of a TV

HTML = The TV

The physical device. It exists, it has structure.

CSS = Picture Settings

Brightness, color, contrast. How it LOOKS.

JavaScript = Remote Control

Makes things HAPPEN. Change channel, adjust volume.

üì¶ Variables: Boxes with Labels


                

                    
What is a Variable?

                    

                        A variable is like a labeled box where you can store information.
                        You can put things in, take things out, and change what's inside!
                    

                


                

                    
üéÅ Real-Life Example

                    
Imagine you have a box labeled "FAVORITE_FOOD".

                    
Inside, you put a note that says "Pizza".

                    
Anytime you need to remember your favorite food, you look in that box!

                    
Later, you can change it to "Sushi" by replacing the note.

                


                
üíª In Code


                

                    
JavaScript

                    
// Create a box called "myName" and put "Sofia" inside
let myName = "Sofia";

// Create a box for age
let age = 25;

// Create a box for favorite color
let favoriteColor = "blue";

// Show what's in the box
console.log(myName);  // Shows: Sofia

                


                

                    
üí° Understanding the Syntax

                    
    
                        
  • let = "Create a new box"
    • 
                        
  • myName = The label on the box
    • 
                        
  • = = "Put this inside"
    • 
                        
  • "Sofia" = What goes in the box
    • 
                        
  • ; = End of instruction (period)
    • 
                    

                


                
üîÑ Variables Can Change!


                

                    
JavaScript

                    
let mood = "happy";
console.log(mood);  // Shows: happy

// Later... change the mood!
mood = "excited";
console.log(mood);  // Shows: excited

                


                
üìù Types of Things You Can Store


                

                    

                        
Text (Strings)

                        let name = "Maria";
                        
Use quotes!

                    

                    

                        
Numbers

                        let age = 30;
                        
No quotes needed

                    

                    

                        
True/False

                        let isHappy = true;
                        
Called "boolean"

üç≥ Functions: Recipes You Can Reuse


                

                    
What is a Function?

                    

                        A function is like a recipe - a set of instructions you can use over and over.
                        Write once, use many times!
                    

                


                

                    
üçû Recipe Analogy

                    

                        Recipe: Make Toast
                        
    
                            
  1. Get bread
    2. 
                            
  2. Put in toaster
    3. 
                            
  3. Wait 2 minutes
    4. 
                            
  4. Take out
    5. 
                            
  5. Add butter
    6. 
                        

                        

                            Now you can make toast ANYTIME just by saying "makeToast()"!
                            You don't have to repeat all the steps every time.
                        

                    

                


                
üíª In Code


                

                    
JavaScript

                    
// Define the function (write the recipe)
function sayHello() {
    console.log("Hello!");
    console.log("Welcome!");
    console.log("Nice to meet you!");
}

// Use the function (follow the recipe)
sayHello();  // Runs all 3 console.logs

// Use it again!
sayHello();  // Runs again!

                


                

                    
üí° Understanding Function Syntax

                    
    
                        
  • function = "I'm creating a recipe"
    • 
                        
  • sayHello = Name of the recipe
    • 
                        
  • () = "Ingredients go here" (empty for now)
    • 
                        
  • {...} = The actual instructions
    • 
                        
  • sayHello() = "Execute the recipe!"
    • 
                    

                


                
üéõÔ∏è Functions with Inputs (Parameters)


                

                    
JavaScript

                    
// Function that needs an INPUT
function greet(name) {
    console.log("Hello, " + name + "!");
}

// Use it with different inputs
greet("Maria");   // Shows: Hello, Maria!
greet("Jo√£o");    // Shows: Hello, Jo√£o!
greet("Sofia");   // Shows: Hello, Sofia!

                


                

                    
üçû Toast Recipe with Options

                    

                        Recipe: Make Toast (with slices parameter)
                        
    
                            
  1. Get [numberOfSlices] slices of bread
    2. 
                            
  2. Put in toaster
    3. 
                            
  3. Wait 2 minutes
    4. 
                            
  4. Take out
    5. 
                            
  5. Add butter
    6. 
                        

                        

                            makeToast(1) ‚Üí Makes 1 slice

                            makeToast(4) ‚Üí Makes 4 slices!
                        

                    

                


                
üîô Functions Can Return Values


                

                    
JavaScript

                    
// Function that GIVES BACK a result

    let sum = a + b;
    return sum;  // Give back the answer
}

// Use the result

console.log(result);  // Shows: 8

// Or use it directly
console.log(addNumbers(10, 20));  // Shows: 30

üéØ Interactive Demo: Click Counter


                

                    

                        Click Me!
                    

                    

                        Clicks: 0
                    

                    

                        
                    

                


                
üíª How the Counter Works


                

                    
JavaScript

                    
// Step 1: Create a variable to store the count
let clickCount = 0;

// Step 2: Grab the elements we need
let area = document.getElementById('clickArea');
let counter = document.getElementById('clickCounter');

// Step 3: Listen for clicks
area.addEventListener('click', function() {
    // Add 1 to the count
    clickCount = clickCount + 1;
    
    // Update the number on screen
    counter.textContent = clickCount;
    
    // Change color after 10 clicks
    if (clickCount > 10) {
        area.style.background = 'gold';
    }
    
    console.log('Total clicks:', clickCount);
});

                


                

                    
üí° Understanding addEventListener

                    

                        Think of it like this:

                        "Hey computer, LISTEN to this element. When someone CLICKS it, run this function!"
                    

                    
    
                        
  • addEventListener = "Start listening"
    • 
                        
  • 'click' = What to listen for
    • 
                        
  • function() {...} = What to do when it happens
    • 
                    

                


                
üî¢ Understanding the Counter Logic


                

                    
Step by Step

                    
// Start with 0
let clickCount = 0;

// First click:
clickCount = clickCount + 1;  // 0 + 1 = 1
counter.textContent = 1;       // Show "1"

// Second click:
clickCount = clickCount + 1;  // 1 + 1 = 2
counter.textContent = 2;       // Show "2"

// Third click:
clickCount = clickCount + 1;  // 2 + 1 = 3
counter.textContent = 3;       // Show "3"

// And so on...

                


                

                    
üìä Shortcut Way to Add

                    
Instead of clickCount = clickCount + 1;

                    
You can write: clickCount++;

                    
It means the exact same thing! Programmers love shortcuts.

üíª How Drawing Works


                

                    
The Magic: Canvas Element

                    

                        <canvas> is a special HTML element where we can draw using JavaScript!
                        Think of it as a digital canvas where code becomes art.
                    

                


                
üìù Step 1: Setup


                

                    
JavaScript

                    
// Grab the canvas
let canvas = document.getElementById('drawCanvas');

// Get the "context" - our drawing tool
let ctx = canvas.getContext('2d');

// Variables to track drawing
let isDrawing = false;
let currentColor = 'black';

                


                

                    
üí° What is "ctx"?

                    

                        ctx stands for "context" - think of it as your paintbrush!

                        It's the tool that actually draws on the canvas.
                    

                


                
üìù Step 2: Mouse Events


                

                    
JavaScript

                    
// When mouse is PRESSED
canvas.addEventListener('mousedown', function(event) {
    isDrawing = true;  // Turn drawing ON
    
    // Start drawing from mouse position
    let x = event.offsetX;
    let y = event.offsetY;
    ctx.beginPath();
    ctx.moveTo(x, y);
});

// When mouse MOVES
canvas.addEventListener('mousemove', function(event) {
    // Only draw if mouse is pressed
    if (isDrawing) {
        let x = event.offsetX;
        let y = event.offsetY;
        
        // Draw line to new position
        ctx.lineTo(x, y);
        ctx.strokeStyle = currentColor;
        ctx.lineWidth = 5;
        ctx.stroke();
    }
});

// When mouse is RELEASED
canvas.addEventListener('mouseup', function() {
    isDrawing = false;  // Turn drawing OFF
});

                


                

                    
üñäÔ∏è Think of it Like:

                    
    
                        
  • mousedown = Put pen on paper
    • 
                        
  • mousemove = Move pen while touching paper
    • 
                        
  • mouseup = Lift pen off paper
    • 
                        
  • isDrawing = Is pen touching paper? (true/false)
    • 
                    

                


                
üìù Step 3: Helper Functions


                

                    
JavaScript

                    
// Change drawing color
function setColor(newColor) {
    currentColor = newColor;
    console.log('Color changed to:', newColor);
}

// Clear everything
function clearCanvas() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    console.log('Canvas cleared! üßπ');
}

                


                
üé® Understanding Drawing Commands


                

                    

                        
beginPath()

                        
Start a new drawing path (pick up pen)

                    

                    

                        
moveTo(x, y)

                        
Move to position without drawing

                    

                    

                        
lineTo(x, y)

                        
Draw line from last position

                    

                    

                        
stroke()

                        
Actually draw the line!
