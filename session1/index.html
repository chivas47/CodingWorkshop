💡 What is Code?

Before we write a single line, let's understand what we're actually doing.

Code = Instructions for Computers

Computers are very fast, but also very dumb. They only do exactly what you tell them. Code is the language we use to give those instructions.

Just like a recipe tells a chef how to make a cake, code tells a computer what to do.

📦 Variables

A variable is a named box where you can store information.

Imagine Labeled Boxes

You have a box labeled FAVORITE_FOOD. You put a note inside that says "Pizza". Anytime you need to remember your favorite food, you look in that box.

Later you can change "Pizza" to "Sushi" — the box stays, only the content changes. That's why it's called a variable — it varies!

Step by Step — Creating a Variable

Write the keyword let

This tells JavaScript: "I want to create a new box."

Give the box a name

Names can't have spaces. Use camelCase: myName not my name. Make it descriptive!

Write = (equals sign)

This means "put this inside". It's NOT asking "is it equal" — it's saying "put this value in the box".

Write the value

Text goes inside quotes: "Sofia" — Numbers don't: 25

End with ;

The semicolon is like a full stop — it tells JavaScript the instruction is finished.

JavaScript
// Full example — reads like a sentence:
// "Create a box called myName and put Sofia inside"
let myName = "Sofia";

let age = 25;
let favoriteColor = "blue";
let isHappy = true;

// See what's inside:
console.log(myName);  // → Sofia

Three Types of Values

📝 Text (String)

let name = "Maria";

Always use quotes. Can be any text, even emojis!

🔢 Number

let age = 30;

No quotes. Works with decimals too: 3.14

✅ True / False

let isHappy = true;

Called a "boolean". Only two values: true or false

🍳 Functions

A function is a recipe — a set of instructions you write once and use many times.

The Problem Functions Solve

Imagine you need to say "Hello! Welcome! Nice to meet you!" 100 times on your website. Would you write those 3 lines 100 times? No!

You write them once inside a function, then just call that function 100 times.

Step by Step — Creating a Function

Write the keyword function

"I am creating a recipe."

Give the function a name

Same rules as variables: camelCase, no spaces. Pick an action name like sayHello or makeRed.

Add parentheses ()

Inputs (ingredients) go here. Leave empty if it doesn't need any inputs yet.

Add curly braces { }

Everything between { and } is the recipe — the instructions that run when you call the function.

Call it with functionName()

Writing the name with () at the end executes the function — it runs all the instructions inside.

JavaScript
// DEFINE the function (write the recipe once)
function sayHello() {
    console.log("Hello!");
    console.log("Welcome!");
    console.log("Nice to meet you!");
}

// CALL the function (follow the recipe)
sayHello();  // Runs all 3 lines above
sayHello();  // Runs again!
sayHello();  // And again!

📥 Functions with Inputs (Parameters)

The name in parentheses is called a parameter — it's a variable that holds whatever value you pass in when you call the function.

🍞 The Toast Analogy

Recipe: makeToast(numberOfSlices)

1. Get [numberOfSlices] of bread → 2. Put in toaster → 3. Wait → 4. Take out

Now: makeToast(1) makes 1 slice. makeToast(4) makes 4 slices! Same recipe, different result.

🔙 Functions That Return a Value

JavaScript
function addNumbers(a, b) {
    let sum = a + b;
    return sum;  // ← send the result back
}

let result = addNumbers(5, 3);
console.log(result);  // → 8

How Does It Work? Step by Step

Every element on a webpage has an ID — a unique name so JavaScript can find it. Our box has id="demoBox". Here's how we grab it and change it:

Find the element — document.getElementById()

This is like asking: "Hey, page — give me the element with this ID." The answer goes into a variable.

let box = document.getElementById('demoBox');

Change a style property

Use .style.propertyName to change CSS from JavaScript.

box.style.background = 'red';
box.style.width      = '400px';
box.style.borderRadius = '50%';

Change the content

Use .textContent to change the text or emoji inside.

box.textContent = '🎉';

Full Example: makeRed()

JavaScript
function makeRed() {
    // Step 1: grab the element
    let box = document.getElementById('demoBox');

    // Step 2: change its background
    box.style.background = 'red';

    console.log('Box is now red! 🔴');
}

🖱️ Practice 2: Events

Events are things that happen — clicks, movement, key presses. We can listen and react!

What is an Event?

Every time something happens on a webpage — someone clicks, moves the mouse, presses a key — JavaScript can "hear" it and run a function in response.

We call this event listening. Think of it as setting a trap: "When THIS happens, do THAT."

🎯 Demo — Click Counter

Click Me!
0
clicks

Building the Counter — Step by Step

Create a variable to keep track of the count

This variable is our "memory" — it stores how many times we've clicked.

let clickCount = 0;  // Start at zero

Grab both elements

let area    = document.getElementById('clickArea');
let counter = document.getElementById('clickCounter');

Add an event listener

area.addEventListener('click', function() {
    clickCount++;
    counter.textContent = clickCount;
});

Understanding clickCount++

The ++ is a shortcut for clickCount = clickCount + 1. It adds 1 to the variable. Programmers love shortcuts!

Optional: React when reaching a milestone

if (clickCount === 10) {
    area.style.background = 'gold';
    area.textContent = '🎉 10 clicks!';
}

⚡PHASE 3 — JavaScript: Making It Work

Add a <script> tag before </body>

⚠️ Why at the bottom?

If the script runs before the HTML loads, getElementById won't find the canvas — it doesn't exist yet!

Grab the canvas and get the drawing context

let canvas = document.getElementById('myCanvas');
let ctx = canvas.getContext('2d');

💡 What is ctx?

ctx is your paintbrush — all drawing commands go through it.

Create state variables

let isDrawing = false;
let currentColor = 'black';

Listen for mousedown

canvas.addEventListener('mousedown', function(e) {
    isDrawing = true;
    ctx.beginPath();
    ctx.moveTo(e.offsetX, e.offsetY);
});

Listen for mousemove

canvas.addEventListener('mousemove', function(e) {
    if (isDrawing === false) { return; }
    ctx.lineTo(e.offsetX, e.offsetY);
    ctx.strokeStyle = currentColor;
    ctx.lineWidth   = 5;
    ctx.lineCap     = 'round';
    ctx.stroke();
});

👉 Save and refresh. Click and drag on the canvas — you should see a line appear!

Listen for mouseup and mouseleave

canvas.addEventListener('mouseup',    function() { isDrawing = false; });
canvas.addEventListener('mouseleave', function() { isDrawing = false; });

Write setColor() and clearCanvas()

function setColor(newColor) {
    currentColor = newColor;
}
function clearCanvas() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
}

👉 Click a colour button and draw. The Clear button should wipe the canvas.

How Each Effect Is Built

💨 Spray Paint — Random Dots

Instead of one line, we scatter many tiny dots around the cursor using Math.random().

for (let i = 0; i < density; i++) {
    let angle  = Math.random() * Math.PI * 2;
    let radius = Math.random() * brushSize;
    let x = mouseX + Math.cos(angle) * radius;
    let y = mouseY + Math.sin(angle) * radius;
    ctx.fillRect(x, y, 2, 2);
}

🌈 Rainbow Trail — Color Math

We cycle a hue variable from 0 to 360. CSS HSL colors let us use this directly.

let hue = 0;
function draw(x, y) {
    hue = (hue + 3) % 360;
    ctx.strokeStyle = `hsl(${hue}, 100%, 50%)`;
}

🦋 Symmetry — Mirror Drawing

Every stroke you make is mirrored across the canvas centre using math.

let centerX = canvas.width / 2;
let mirrorX = centerX - (x - centerX);
